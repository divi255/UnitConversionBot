I can convert one units to another. Just type something like "100 ft to m" and I'll convert it for you.

I can convert one units to another. Just type something like "100 ft to m" and I'll convert it for you.

/setcommands
convert - Convert one unit to another (e.g., "10 ft to m")
help - About this bot

/setinline
/setnoinline
